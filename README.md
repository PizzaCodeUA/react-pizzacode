# react-pizzacode
Build Your First React.js App

## Dependencies
- react
- babel
- webpack

## Install
1. `npm init`
2. `npm install --save react`
3. `npm install --save-dev babel-loader`
4. `npm install webpack -g`

## Run
1. `webpack -w`
2. check `bundle.js` in `public` folder
2. open `index.html`
